Portafolio Web — Nicolás González

Descripción 
Sitio web personal y portafolio que presenta información sobre el autor, sus habilidades y proyectos.
Este repositorio contiene la versión estática de una página HTML con estilos CSS que muestra las secciones Sobre mí, Habilidades, Proyectos y Contacto, además de un formulario de contacto y enlaces a repositorios de GitHub.

Contenido del repositorio
-index2.html — Página principal del portafolio (estructura y contenido).
-style.css — Estilos y diseño visual del sitio.
-assets/ — Imágenes y recursos (logo, avatar, iconos, etc.).

Tecnologías usadas
-HTML5 — Estructura semántica de la página (secciones, artículos, nav, footer).
-CSS3 — Estilos, layout con Flexbox y Grid, y componentes visuales (tablas, tarjetas, formulario).
.Recursos web — Fuentes desde Google Fonts y recursos estáticos en la carpeta assets/.

Como ejecutar localmente:
1. Clona el repositorio: git clone https://github.com/Nialgoag-GH/tu-repo.git
2. Entra en la carpeta del proyecto: cd tu-repo
3. Abre index2.html con tu navegador o Live Preview en VSCode
